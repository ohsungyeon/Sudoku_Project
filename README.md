# 🧩 Sudoku Project

Python과 Streamlit을 활용하여 구현한 스도쿠 게임입니다.

스도쿠는 가로, 세로, 그리고 3×3 영역 안에 1부터 9까지의 숫자가 중복되지 않도록 채워 넣는 논리 퍼즐 게임입니다. 사용자는 웹 화면에서 숫자를 입력하며 퍼즐을 완성할 수 있습니다.

## 🎯 프로젝트 목표

- 스도쿠 게임 로직 구현
- 퍼즐 생성 및 정답 검증 기능 구현
- Streamlit을 활용한 웹 기반 게임 개발

## 📋 규칙

- 각 행에는 1부터 9까지의 숫자가 중복되지 않게 채워져야 합니다.
- 각 열에는 1부터 9까지의 숫자가 중복되지 않게 채워져야 합니다.
- 각 3×3 영역에는 1부터 9까지의 숫자가 중복되지 않게 채워져야 합니다.

## ✨ 주요 기능

- 랜덤 스도쿠 퍼즐 생성
- 숫자 입력 및 수정
- 행, 열, 3×3 영역 유효성 검사
- 정답 확인 기능
- 게임 완료 판정
- 웹 기반 UI 제공

## ⚙️ 함수 설명

- `initialize_board()` : 빈 스도쿠 보드를 생성합니다.
- `print_board()` : 스도쿠 보드를 출력합니다.
- `solve_sudoku()` : 백트래킹 알고리즘을 이용해 스도쿠를 해결합니다.
- `generate_sudoku()` : 완성된 스도쿠 보드를 생성한 후 일부 칸을 제거하여 퍼즐을 생성합니다.
- `play_sudoku()` : 스도쿠 게임을 실행하고 사용자 입력을 처리합니다.

## 🚀 실행 방법

### 1. 라이브러리 설치

```bash
pip install streamlit
```

### 2. 프로그램 실행

```bash
streamlit run app.py
```

### 3. 브라우저에서 게임 플레이

실행 후 생성된 로컬 주소(`http://localhost:8501`)로 접속하여 게임을 플레이할 수 있습니다.
